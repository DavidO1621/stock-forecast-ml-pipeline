# Stock Price Forecasting ML Pipeline

> An end-to-end time-series data pipeline analyzing five years of 
> historical AAPL stock data using yfinance, domain-specific feature 
> engineering, a comparative ML benchmarking framework (Linear Regression, 
> Random Forest, XGBoost, and LSTM), and LLM-powered analyst commentary 
> via the Anthropic Claude API — built to answer: *which model best 
> predicts stock price movement, and what drives significant prediction 
> errors on specific dates?*
